# Contributing to SkyMarshal

Thank you for considering contributing to SkyMarshal! We welcome all contributions that help improve this traffic monitoring system.

## How to Contribute

1.  **Fork the Repository**: Create your own copy of the project.
2.  **Create a Branch**: Use a descriptive name for your branch (e.g., `feat/add-gui-support`).
3.  **Make Changes**: Implement your features or bug fixes. Ensure code follows PEP 8 standards.
4.  **Test**: Verify your changes against sample videos.
5.  **Submit a Pull Request**: Provide a clear description of what your changes do.
